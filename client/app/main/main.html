What is Launch?

Reactor Prep is a 4 week part-time course offered by Tech People towards beginners with little-to-no experience with computer programming. Over the course of 20 classes, you’ll learn the fundamentals of HTML5, CSS3, and Javascript - also known as Front-End Web Development. By the end of this course, you’ll be able to create static websites, build single page web applications, and have a good sense of whether or not programming is right for you.

This course is being taught by Robert Raleigh, a Tech People co-founder and experienced developer and instructor. The curriculum is comprised of code-alongs, projects, and brief lectures. Students will quickly become familiar with tools including the code editor, browser console, and terminal. If you have ever thought about becoming a software engineer, this course provides a quick and effective way to learn the basics.
